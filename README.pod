=head1 Group-Git

Tool for operating on many git repositories at once. The repositories can be
managed by Github, Bitbucket, Gitosis or not at all. If managed uncloned
repositories will be automatically cloned.

=head1 INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install
